00022 /*
00023  * Modules are loaded into environments. By default, search path is given by
00024  * variable named dfsch:*load-path*, with dfsch:*modules* being list of 
00025  * already loaded modules.
00026  * 
00027  * Load path list can contain either pathnames as strings or callable objects
00028  * callable object is passed and environment and name of module to load,
00029  * it is expected to do any processing internally and return non-nil value if 
00030  * successful.
00031  *
00032  * Set of built-in modules and on-disk module formats is easily extensible, 
00033  * but currently only by modifying src/load.c.
00034  */
