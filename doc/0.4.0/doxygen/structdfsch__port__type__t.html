dfsch_port_batch_read_t dfsch_port_type_t::batch_read

Fast read of one byte that must be inside batch_read_start()/batch_read_end() calls. When this is not implemented it will be emulated by means of read_buf(). General idea is that this will use some mechanism like unlocked_stdio(3).
